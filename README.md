# PDND-keygen
Small Powershell script to generate RSA keys for PDND authentication

## Usage

```
> .\pdnd-keygen.ps1 my-keys

✔️ PEM generated
writing RSA key
✔️ PUB generated
✔️ PRIV generated
✔️ Keys my-keys successfully generated in 1 sec
```
